# AI Knowledge Assistant

An AI-powered conversational assistant built using Python and OpenRouter.

## Current Features

- Interactive AI chatbot
- Conversation memory
- System prompt
- OpenRouter API integration
- Environment variable support
- Modular project structure

## Tech Stack

- Python
- OpenRouter
- OpenAI Python SDK
- dotenv

- PDF & DOCX document support
- ChromaDB

- Agentic AI
- Machine Learning document classification
- Analytics Dashboard
- FastAPI Backend
- React Frontend
